# 身元保証書機能

## 運用仕様

- 申請人1名につき身元保証書1通を作成する。
- 複数申請人を1通にまとめる機能（「ほか○名」）は実装しない。
- 身元保証人氏名欄には、所属・肩書・氏名をまとめて入力できる。
- 初期値は `立命館大学　国際業務課課長　田中 猛`。
- 身元保証人の職業は任意入力。
- 身元保証人の生年月日を入力すると、書類作成日現在の年齢を自動計算する。
- 公館種別は「未選択」「大使館」「総領事館」から選択する。
- 未選択のままPDFを出力し、提出時に手書きでチェックしてもよい。
- FAX番号は任意入力。

## 出力

- 選択中の申請人の身元保証書PDF
- 選択中の申請人について、招へい理由書と身元保証書をまとめたZIP
- Excelに読み込んだ全申請人について、1名1通の身元保証書をまとめたZIP

## 実装上の注意

身元保証書はCanvasで描画し、既存のPDF出力処理を再利用してA4 PDFに変換する。座標は今後、実際の出力見本を確認しながら微調整する。
