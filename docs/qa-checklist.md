# 身元保証書 QA checklist

- [ ] Excelから選択した申請人の氏名、国籍、職業、性別、生年月日、年齢が正しく表示される
- [ ] 保証人の所属・肩書・氏名が1つの欄に表示される
- [ ] 職業空欄でも出力できる
- [ ] 保証人生年月日から書類作成日現在の年齢が計算される
- [ ] 公館種別未選択で両方のチェックが空欄になる
- [ ] 大使館選択で大使館のみチェックされる
- [ ] 総領事館選択で総領事館のみチェックされる
- [ ] 1申請人につき1PDFになる
- [ ] 「ほか○名」は出力されない
- [ ] 個別PDF、2書類ZIP、全員分ZIPが開ける
- [ ] A4で印刷した際に文字切れや重なりがない
