# UI notes

The guarantee letter settings appear above the preview area.

Fields:

- guarantor organisation, title and name
- optional occupation
- date of birth
- mission type: none, embassy, consulate
- optional guarantor fax
- optional contact fax

The selected applicant section provides buttons for:

- invitation reason PDF
- guarantee letter PDF
- both documents as ZIP

Batch export keeps the operational rule of one guarantee letter per applicant.
