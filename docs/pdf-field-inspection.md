## Mapping conclusions

- The PDF template file was inspected by AcroForm widget rectangle, not by field order alone.
- The fillable template at `public/templates/shouhei-riyusho.pdf` has been inspected and the mapping in `src/lib/pdfFieldNames.ts` is complete.
- `documentDateEra` is intentionally not part of the semantic model: the template prints the visible `令和` text and has only year/month/day widgets.
- `additionalApplicantsCount` is the `ほか N 名` field. The MVP always has exactly one applicant, so it must remain blank and must not be treated as gender.
- `diplomaticMission` maps to `公館名` at x=102.24, y=713.24, width=87.97. This widget is the blank after the printed `在` and before the printed `日本国`; therefore the value is `上海`, not `在上海日本国総領事館`.
- The inviter address is a single widget (`住所1` at x=189.48, y=608.48, width=297.84), so the semantic model remains `inviterAddress`.
